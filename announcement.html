关于开展2025年度政府部门绩效考核的通知

发布部门：人事部
发布人：王人事
发布日期：2025-03-20
浏览次数：158
通知 重要 绩效考核

各部门：

为全面推进我市政务工作高质量发展，根据市委市政府有关工作部署，现就开展2025年度政府部门绩效考核工作通知如下：

一、考核目的

通过绩效考核，客观评价各部门工作成效，强化责任意识，提高行政效能，为加强和改进政府工作提供依据。

二、考核范围

市级各部门、各直属机构。

三、考核内容

  1. 重点工作完成情况；
  2. 日常工作执行效率；
  3. 创新工作开展情况；
  4. 群众满意度评价；
  5. 廉政建设情况。

四、考核方式

采取定量与定性相结合、日常考核与年度考核相结合、部门自评与领导评价相结合的方式。具体包括：

  1. 部门自评（30%）；
  2. 数据考核（30%）；
  3. 实地核查（20%）；
  4. 社会评价（20%）。

五、时间安排

  1. 4月1日-4月15日：各部门准备自评材料；
  2. 4月16日-4月30日：考核组实地核查；
  3. 5月1日-5月15日：汇总评价，形成考核结果；
  4. 5月20日前：向市政府报告考核结果，并向各部门反馈。

六、工作要求

  1. 各部门要高度重视绩效考核工作，认真组织实施；
  2. 考核资料要真实准确，不得弄虚作假；
  3. 人事部门要加强对考核工作的监督，确保考核工作公平公正；
  4. 考核结果将作为部门评优评先、领导干部提拔任用的重要依据。

特此通知。

人事部

2025年3月20日
